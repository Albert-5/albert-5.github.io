2019-02-18

最近发现使用Cisco AnyConnect在Win10上系统上连接VPN时，会出现anyconnect was not able to establish a connection to the specified secure gateway问题导致无法成功连接VPN，错误提示如下所示

problem:

AnyConnect was not able to establish a connection to the specified secure gateway. Please try connecting again.

其实解决方法也是很简单的，只需要：

此电脑鼠标右击 ->管理 ->服务于应用程序 -> 服务

-> 禁用图中的 Internet Connection Sharing即可
